# iris16688.github.io
客户记账系统

## 功能特点

- 客户管理：添加、查看和删除客户信息
- 欠款记录：记录客户欠款，包括金额、日期、事由等
- 还款记录：记录客户还款，支持多种还款方式
- 数据统计：直观展示总欠款、已还款、逾期等数据
- 历史记录：查看所有交易历史
- 搜索功能：快速查找客户和交易记录
- 打印收据：为还款记录生成打印收据
- 离线使用：支持离线访问和数据存储
- PWA支持：可安装为本地应用使用

## 如何使用

### 在线使用

1. 直接打开 `index.html` 文件即可使用系统
2. 无需安装任何依赖，所有功能都在浏览器中运行

### 离线使用

系统已配置为 PWA (Progressive Web App)，支持离线使用：

1. 在 Chrome 浏览器中打开系统
2. 点击地址栏右侧的 "安装" 图标
3. 按照提示将应用安装到本地
4. 安装后可在没有网络连接的情况下使用

### 数据存储

- 所有数据都存储在浏览器的 localStorage 中
- 数据仅保存在当前浏览器中，不会上传到服务器
- 建议定期备份数据以防丢失

## 浏览器兼容性

- Chrome (推荐)
- Firefox
- Safari
- Edge

## 技术栈

- HTML5
- Tailwind CSS
- JavaScript
- Chart.js
- PWA (Progressive Web App)

## 注意事项

1. 首次使用时，系统会自动创建一些示例数据
2. 清除浏览器缓存可能会导致数据丢失
3. 如需在多台设备间同步数据，请手动导出和导入数据
4. 建议使用 Chrome 浏览器以获得最佳体验
